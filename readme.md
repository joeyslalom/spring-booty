## What is this

* Java Web application: Spring Boot
  * https://projects.spring.io/spring-boot/
* OpenAPI (aka Swagger) integration via SpringFox
  * http://springfox.github.io/springfox/

## Run locally

Build
* ./gradlew build

Run:
* ./gradlew bootRun

Swagger:
* http://localhost:8080/swagger-ui.html
